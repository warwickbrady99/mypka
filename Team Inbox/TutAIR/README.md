# TutAIR Inbox

TutAIR is the GCSE learning intake lane for myPKA.

Use this folder for educational YouTube links, copied transcript text, textbook extracts, lesson notes, revision-guide snippets, and pasted learning material that should become ADHD-friendly GCSE revision resources.

## Folder Pattern

Captures live under date folders:

```text
Team Inbox/TutAIR/YYYY/MM/
```

Example:

```text
Team Inbox/TutAIR/2026/07/2026-07-09-gcse-biology-cell-division.md
```

## What A Capture Should Include

Each capture should keep the raw learning source available and include:

- subject
- topic
- possible exam board
- source URL, if there is one
- date captured
- confidence level
- source text or transcript

## Exam Board Safety Rule

Do not treat a possible exam board as confirmed unless there is evidence from an official specification, teacher, school document, exam timetable, or confirmed course source.

Use wording like:

- `possible_exam_board: AQA`
- `exam_board_status: unconfirmed`
- `exam_board_evidence: none`

When the exam board is confirmed, link the capture or processed note to the relevant source of truth, such as `[[exam-board-map]]`.

## Processing Path

TutAIR follows this shape:

```text
raw learning source -> TutAIR inbox capture -> ADHD-friendly learning note -> later dashboard
```

Raw captures stay here as evidence. Processed learning notes and durable GCSE knowledge should be created through [[WS-005-process-tutair-captures]].
