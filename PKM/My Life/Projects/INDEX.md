# Projects - Index

Bounded efforts with a finish line — a set of deliverables you cross off when finished. A Project is one of the two shapes a Goal's carrier can take, the sibling of a Habit: use a Project when the work has an end, a Habit when it is an open-ended rhythm.

This folder is a **thinking space, not a task tracker.** A Project file holds the shape of the effort — what "done" means, what it delivers, what it is teaching you — not a checklist of tasks. The granular to-dos live in your task manager; this file holds the why and the finish line.

## The carrier role

A Goal names **one** carrier. When that carrier is a Project, the Project is the Goal's sole vehicle — there is no parallel Habit also carrying it.

Example: the Goal *"lose 20 kg"* can be carried by a Project (an 8-week program with an end date) **or** by a Habit (3 workouts a week) — never both. When a Project is the carrier, it owns the delivery.
