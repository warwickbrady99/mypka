# Student Guide

This guide is for using myPKA as a simple study helper for school, revision, tutoring, and exams.

You do not need to make it perfect. You only need to make it useful.

## What This System Is For

Use this system to keep track of:

- What subjects you study.
- Which exam board and course you are doing.
- What topics are in each subject.
- What you find difficult.
- What you should revise next.
- Which resources match your actual course.
- How AI should tutor you.

## First Setup Steps

Start by filling in only what you know.

- [ ] Open `docs/exam-board-map.md`.
- [ ] Check each subject row.
- [ ] Add any exam board details you already know.
- [ ] Mark anything uncertain as `needs confirmation`.
- [ ] Ask teachers or check school documents for missing details.
- [ ] Add links to official specifications or revision guides when confirmed.
- [ ] Do not guess.

## What To Capture For Each Subject

For each subject, try to capture:

- Subject name.
- Qualification type.
- Exam board.
- Exact course or specification.
- Tier, if relevant.
- Topics or modules.
- Links to specifications.
- Links or names of revision guides.
- Teacher recommendations.
- What you struggle with.
- What to revise next.
- How AI tutoring should help.

## Quick Daily Use

On a normal day, keep it small.

1. Pick one subject.
2. Pick one topic.
3. Write your confidence score from 0 to 4.
4. Revise for 20 minutes.
5. Do a few questions.
6. Write one sentence about what was hard.
7. Choose the next tiny task.

## Good Notes Look Like This

Weak note:

```text
Science is confusing.
```

Useful note:

```text
Science: I understand food chains, but I get confused when energy transfer is shown as a percentage. Revise this next with 5 practice questions.
```

Weak note:

```text
English essay bad.
```

Useful note:

```text
English: I know the quote, but I do not explain how it links to the question. Practise one paragraph using point, evidence, explanation, link.
```

## How To Use AI Well

Ask AI to tutor you, not just answer for you.

Good requests:

- Explain this topic like I am learning it for the first time.
- Ask me one question at a time.
- Give me a hint, not the answer yet.
- Mark my answer and tell me how to improve it.
- Give me 3 easier questions, then 3 harder ones.
- Help me spot the pattern in my mistakes.
- Turn this mistake into a revision task.

Avoid requests like:

- Do my homework for me.
- Write my coursework.
- Give me answers only.
- Guess my exam board.

## What To Do When You Are Stuck

Use this sentence:

```text
I am stuck on <topic>. I think the problem is <specific struggle>. Please explain it simply, ask me questions, and help me practise.
```

Examples:

- I am stuck on simultaneous equations. I think the problem is choosing which method to use.
- I am stuck on history source questions. I think the problem is explaining usefulness.
- I am stuck on computer science trace tables. I think the problem is following variable changes.

## What Dad Or A Tutor Can Help With

They can help you:

- Confirm exam boards and course details.
- Find the right specification or revision guide.
- Choose a realistic weekly revision plan.
- Notice patterns in what you struggle with.
- Check that AI tutoring is helping you understand.
- Keep the system simple.

They should not need to fill everything in for you. The best version is one you actually understand and use.

## The Rule

If the system gets too complicated, make it smaller.

One confirmed subject detail, one clear struggle, and one next revision task is already progress.
