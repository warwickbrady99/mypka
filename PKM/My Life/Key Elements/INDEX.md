# Key Elements - Index

The load-bearing walls of your life — Health, Family, Career, Wealth, Creative work. Stable dimensions that shift only with major life changes. Aim for roughly **5 to 15** files: fewer and you are missing walls; more and you have Topics in disguise.

## Every Goal anchors here

Key Elements are where Goals attach. **Every Goal anchors to exactly one Key Element** — never to a Project, never to a Topic. The Key Element is the stable "why this matters" behind the Goal.

Example: the Goal *"lose 20 kg"* anchors to [[health]]. Health is the wall; the Goal is the push against it.

## Topics graduate into here

A Key Element is also a destination, not just a starting point. A [[PKM/My Life/Topics/INDEX|Topic]] that crystallizes into a measurable pursuit **graduates into a Key Element** — and only after that promotion can a Goal anchor to it.

Example: *French* starts as a Topic. When it sharpens into a measurable pursuit ("reach B2 fluency"), it graduates into a Key Element. Only then can a Goal anchor to it. The reverse also holds: a Key Element that goes cold — no Goals anchored, no active push — gets archived back down rather than posing as a wall.

## Cross-links

- A Key Element is the anchor for every [[PKM/My Life/Goals/INDEX|Goal]] that serves it — exactly one Goal-to-Key-Element link per Goal.
- Key Elements are the slowest-changing layer; [[PKM/My Life/Topics/INDEX|Topics]] graduate into one once they crystallize.
- A Key Element is a lens, not a tracker — no streaks, no numbers to optimize.
