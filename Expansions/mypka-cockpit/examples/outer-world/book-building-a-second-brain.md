# A book on organizing what you save

## Summary
A practical method for converting a backlog of saved articles into a connected,
searchable library — by deciding the connection at save time, not later.

## Clip
> The moment to file something is the moment you save it. A link you'll "organize
> later" is a link you've already lost.
> — Priya Nadel
